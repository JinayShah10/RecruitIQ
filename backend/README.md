# RecruitIQ Backend

This directory houses the backend service for RecruitIQ, an AI-powered Applicant Tracking System (ATS).

## Technology Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **Language**: TypeScript

## Folder Structure
- `src/config/` - Application and external service configurations
- `src/controllers/` - Request handling layer
- `src/middleware/` - Express middleware functions
- `src/models/` - Database models and schemas
- `src/routes/` - API route definitions
- `src/services/` - Business logic layer
- `src/utils/` - Reusable helper functions
- `src/validators/` - Input validation logic
- `src/types/` - TypeScript interfaces and custom types
- `src/app.ts` - Express application configuration entry point
- `src/server.ts` - Application startup entry point
